YouTube Ad Revenue Prediction App

This project predicts **YouTube Ad Revenue** using Machine Learning based on video performance metrics such as views, likes, comments, watch time, and subscriber count.

The model is trained using regression algorithms and deployed as an interactive web application using Streamlit.

Project Overview
The goal of this project is to analyze video performance data and estimate the potential advertising revenue generated from a YouTube video.

This project demonstrates the complete Machine Learning workflow:
- Data Cleaning
- Feature Engineering
- Model Training
- Model Evaluation
- Deployment using Streamlit

Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib
